Real-Time Sync Setup
Checking…

When configured, every entry, edit, and master upload is pushed instantly to every admin on every device through a Firebase Realtime Database. Without it, this page works in local-only mode (one device, one browser).

One-time setup (5 min)
  1. Go to Firebase Console and sign in with any Google account.
  2. Click Add Project → give it a name (e.g. burgundy-ic) → continue (Google Analytics optional).
  3. In the left menu, click Build → Realtime Database → Create Database.
  4. Choose a location → Start in test mode → Enable.
  5. Copy the URL shown at the top of the database (looks like https://burgundy-ic-default-rtdb.firebaseio.com).
  6. Paste it below and click Save & Connect.
  7. To onboard other admins, share this page with ?fb=<your-url> appended — they'll auto-configure on first visit.

Free tier: 1 GB storage, 10 GB/month transfer — far more than this tracker will ever use.
